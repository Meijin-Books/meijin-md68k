# 第一部　アセンブリ編

『Meijin 68000アセンブリ』（本書 第0〜12章）のサンプルコードです。
メガドライブ上で動く68000アセンブリのプログラムを収録しています。

書籍を読みながら手元で動かすためのものですが、コードだけ読んで動かすこともできます。

## 収録内容

| フォルダ | ファイル | 対応する章 | 内容 |
|---|---|---|---|
| `ch06/` | `hello.asm` | 第6章 | 画面に「HELLO」を表示する |
| `ch11/` | `move.asm` | 第11章 | パッドでスプライトを動かす |

各フォルダには、ビルド用の `build.bat` を同梱しています。

## 必要なもの

Windowsが動くパソコンが一台あれば足ります。実機のメガドライブは要りません。
使う道具は二つで、どちらも無料です。

### 1. vasm（クロスアセンブラ）

書いたコードを、メガドライブが読めるROMファイルに変換します。

- 入手先： http://sun.hasenbraten.de/vasm/
- 必要なのは **`vasmm68k_mot`**（68000系CPU向け・モトローラ構文）です
- ダウンロードした実行ファイルを適当なフォルダ（例：`C:\tools\vasm`）に置き、そのフォルダを環境変数 `PATH` に追加してください

### 2. BlastEm（エミュレータ）

できあがったROMを、画面の中で本物のメガドライブそっくりに動かします。

- 入手先： https://www.retrodev.com/blastem/
- インストール作業は不要です。実行ファイルを置くだけで使えます

> Mac / Linux をお使いの方へ：どちらの道具も各OSに対応しています。`build.bat` の代わりに、下の「手動でビルドする」のコマンドを直接打ってください。

## 使い方

展開したフォルダの中の `ch06` を開き、**`build.bat` をダブルクリック**してください。
ビルドが通れば、BlastEm が起動して「HELLO」が表示されます。`ch11` も同じ手順で、十字キーでスプライトが動きます。

### 手動でビルドする

```
vasmm68k_mot -Fbin -o hello.bin hello.asm
```

できあがった `hello.bin` を、BlastEm にドラッグ＆ドロップすれば動きます。

## うまくいかないときは

**「コマンドが見つかりません」と言われる**
`PATH` の設定が反映されていません。コマンドプロンプトを開き直してください。
それでも駄目なら、`PATH` に登録したフォルダの中に `vasmm68k_mot.exe` が本当にあるか確認します。

**エミュレータが真っ黒の画面のまま**
ビルドは通っているのに何も映らない場合、ベクタテーブルを疑ってください（第6章）。
68000は電源投入時にROMの先頭を読み、そこに書かれた番地へ飛びます。

**エミュレータでは動くのに、実機でだけ動かない**
後期型のメガドライブ（TMSS搭載機）では、起動時に `$A14000` へ `'SEGA'` を書き込む必要があります（第6章）。
本リポジトリのコードは対応済みです。

## 動作を確認した環境

- vasm 2.0x（`vasmm68k_mot`）
- BlastEm 0.6.2
- Windows 11
